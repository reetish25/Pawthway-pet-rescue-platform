Pawthway is a web-based capstone project designed to support pet rescue and animal welfare initiatives by providing a centralized platform for rescue operations, adoption, awareness, and community participation.

Project Overview

The aim of Pawthway is to address real-world challenges faced by animal rescue organizations and individuals by offering a simple, accessible, and user-friendly web platform.

The platform helps connect rescuers, volunteers, donors, and the general public to improve coordination and awareness around animal welfare.

This project was developed as a **capstone project**, focusing on practical problem-solving, frontend development, and user-centric design.


Features

- Pet rescue information and reporting
- Adoption listings and awareness pages
- Donation support for rescue initiatives
- Volunteer registration and participation
- Lost and found pet reporting
- Admin interface for content management
- Awareness and abuse reporting sections


Tech Stack

- HTML5
- CSS3
- JavaScript (Client-side)
