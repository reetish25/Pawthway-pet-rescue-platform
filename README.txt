PAWTHWAY – FULL STACK WEB APP

REQUIREMENTS:
 - Node.js installed
 - MongoDB installed and running locally

SETUP:
 1️⃣ Open terminal inside project folder
 2️⃣ Run:
     npm install
 3️⃣ Start server:
     npm start
 4️⃣ Open browser:
     http://localhost:5000

API ROUTES:
 - POST /api/rescues
 - POST /api/abuse
 - POST /api/adoptions
 - POST /api/donations
 - POST /api/volunteers
 - POST /api/lost-found

ADMIN PAGE:
 - Open:
   http://localhost:5000/admin.html

FRONTEND PAGES:
 index.html
